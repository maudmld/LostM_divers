This repository contains files related to the creation of a canonical data model in TEI format for the ERC LostMa project data. The two ODDs (text and witness) model the data from the release and are illustrated by two XML test files (whose data are partially drawn from the Heurist database). athis.xml serves as an example of the witness ODD; alexandre.xml is an application of the text ODD.

All project data are available on the database hosted on [Heurist](https://heurist.huma-num.fr/heurist/?db=jbcamps_gestes&w=a&q=t%3A101) and on the project's [Github](https://github.com/LostMa-ERC).
